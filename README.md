# Small Office Lab

## Overview

This repository documents my home lab environment, which I use to develop hands-on IT support, system administration, networking, and virtualization skills.

The lab consists of dedicated hardware running Proxmox VE and Hyper-V, allowing me to build and manage virtual machines, practice troubleshooting scenarios, and gain experience with technologies commonly used in enterprise environments.

## Technologies

- Proxmox VE
- Windows Server 2022
  - Active Directory
  - Group Policy
  - DNS
  - DHCP
  - File Server
- Ubuntu Server
  - Web Server
